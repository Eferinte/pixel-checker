## 应用描述
这是一个单页面应用，支持用户拖动、点击上传图片到页面中，页面会展示该图片，并在光标悬浮到图片对应位置时，在光标处渲染一个放大后的切片视图，并显示光标位置的对应的图片像素的像素信息（如rgba值）；

## 功能清单

- 背景使用css实现ps的灰白棋盘效果
- 图片上传+渲染（支持指定最大尺寸、大小的图片，过大时进行必要的性能优化和视图优化）
- 支持图片清空
- 支持图片基本信息读取和展示

## 注意事项
- 实现功能时按component、hook拆分代码，每个文件最好不超过200行
- 对一些静态参数，抽离到config文件中进行管理，如最大图片上传尺寸、最大图片文件大小、支持的图片格式
